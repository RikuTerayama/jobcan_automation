# UI刷新 - 検証用チェックリスト

## 1. デザインシステムの適用確認

### カラー
- [ ] 背景が単色（#121212）になっている（グラデーションなし）
- [ ] コンテナ背景が単色（#1E1E1E）になっている
- [ ] アクセントカラーが#4A9EFFのみ使用されている
- [ ] テキストカラーが適切なコントラスト比（WCAG AA以上）を満たしている

### 角丸
- [ ] コンテナの角丸が8px
- [ ] カードの角丸が6px
- [ ] ボタン・入力フィールドの角丸が4px

### 影
- [ ] コンテナの影が軽い（0 2px 8px rgba(0,0,0,0.1)程度）
- [ ] カードhover時の影が控えめ（0 4px 12px rgba(0,0,0,0.15)程度）
- [ ] ボタンhover時の影が控えめ

### アイコン
- [ ] 絵文字が全てSVGアイコンに置換されている
- [ ] SVGアイコンの線幅が1.5px
- [ ] アイコンのサイズが統一されている（18-20px）

### タイポグラフィ
- [ ] h1が2.5em、h2が2em、h3が1.5em
- [ ] letter-spacingが0.01-0.02emに統一されている
- [ ] text-shadowが削除されている

### 余白
- [ ] セクション間隔が48pxに統一されている
- [ ] フォームグループ間が24pxに統一されている
- [ ] ラベルと入力の間が8pxに統一されている

## 2. アクセシビリティ

### コントラスト比
- [ ] テキストと背景のコントラスト比が4.5:1以上（WCAG AA）
- [ ] 大きなテキスト（18px以上）のコントラスト比が3:1以上
- [ ] アクセントカラー（#4A9EFF）と背景のコントラスト比が十分

**確認方法**: 
- Chrome DevToolsのLighthouseで確認
- WebAIM Contrast Checkerで確認

### キーボード操作
- [ ] Tabキーで全てのインタラクティブ要素にアクセス可能
- [ ] フォーカスインジケーターが明確に表示される（outline: 2px solid #4A9EFF）
- [ ] フォーカストラップがない（モーダル等を除く）
- [ ] Enter/Spaceキーでボタンが動作する

**確認方法**: 
- Tabキーで全ページを操作
- フォーカス可能要素を全て確認

### スクリーンリーダー
- [ ] アイコンにaria-hidden="true"が設定されている
- [ ] 装飾的でない画像にalt属性が設定されている
- [ ] フォームラベルが適切に紐付けられている

## 3. パフォーマンス

### 読み込み速度
- [ ] First Contentful Paint (FCP) が2.5秒以内
- [ ] Largest Contentful Paint (LCP) が2.5秒以内
- [ ] Time to Interactive (TTI) が3.8秒以内

**確認方法**: 
- Chrome DevToolsのLighthouseで確認
- PageSpeed Insightsで確認

### アニメーション
- [ ] スクロールリビールアニメーションが滑らか（60fps）
- [ ] prefers-reduced-motion設定時にアニメーションが無効化される
- [ ] アニメーションが過剰でない（控えめ）

**確認方法**: 
- Chrome DevToolsのPerformanceタブで確認
- システム設定で「動きを減らす」を有効化して確認
