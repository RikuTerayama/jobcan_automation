ベストプラクティスガイド

Jobcan AutoFillを効果的に活用するための、実践的なベストプラクティスをご紹介します。このガイドに従うことで、より安全かつ効率的に勤怠管理を行えます。

1. データ入力の準備

Excelファイルの準備

推奨事項：

  • テンプレートファイルを使用して、必ず正しい形式に従ってください
  • 日付は「YYYY-MM-DD」形式または「YYYY/MM/DD」形式で統一してください
  • 時刻は「HH:MM」形式または4桁の数値（0900など）で入力してください
  • データ入力前に、必ず目視で内容を確認してください

データの検証

重要：入力前に以下の点を必ず確認してください：

  • すべての日付が正しい形式になっているか
  • 開始時刻と終了時刻の関係が論理的に正しいか（開始時刻 < 終了時刻）
  • 空白行や不完全なデータがないか
  • 勤務時間が会社の就業規則に準拠しているか

2. セキュリティとプライバシー

ログイン情報の管理

セキュリティのベストプラクティス：

  • ログイン情報は、個人のセキュアな環境でのみ使用してください
  • 共有PCや公共のWi-Fi環境では使用を避けてください
  • 処理完了後は、ブラウザの履歴をクリアしてください
  • 定期的にパスワードを変更してください

データの保護

本サービスは、一時的な処理のみを行い、個人情報の永続的な保存は一切行いません。しかし、ユーザー側でも以下の点に注意してください：

  • Excelファイルには、勤怠データのみを含めてください（個人情報の混入を避ける）
  • 処理後のファイルは適切に管理してください（不要になったら削除）
  • 他のユーザーとExcelファイルを共有しないでください

3. 効率的な使用方法

処理タイミングの最適化

推奨される処理タイミング：

  • 月次締め後、月初めにまとめて処理する
  • システム負荷の少ない時間帯を選択する（深夜・早朝など）
  • 一度に処理するデータ数を適切に制限する（50件程度が目安）
  • ネットワーク接続が安定している環境で使用する

エラーハンドリング

エラーが発生した場合の対処法：

  1. エラーメッセージを確認し、トラブルシューティングガイドを参照
  2. 問題が解決しない場合は、処理を一度中断し、しばらく待ってから再試行
  3. 同じエラーが繰り返し発生する場合は、Excelファイルの形式を確認
  4. それでも解決しない場合は、お問い合わせから報告

4. 法的・倫理的な考慮事項

就業規則の遵守

重要な注意事項：

  • 所属企業の就業規則やセキュリティポリシーを必ず遵守してください
  • 勤怠管理システムへの不正なアクセスや改ざんは、法令違反となる可能性があります
  • 実際の勤務実態と異なるデータの入力は、就業規則違反となる可能性があります
  • 個人の判断と責任、そして倫理観をもって慎重にご使用ください

適切な使用範囲

本ツールは、以下の目的での使用を想定しています：

  • テンプレートからの勤怠データ入力
  • 過去の勤怠データの再入力（打刻漏れの修正など）
  • 月次締め時の一括データ入力

以下の目的での使用は避けてください：

  • 実際の勤務時間と異なるデータの入力
  • 勤怠改ざんや不正なアクセス
  • 会社のセキュリティポリシーに反する行為

5. トラブルシューティングのヒント

よくある問題と対処法

問題：処理が途中で停止する

対処法：

  • Excelファイルのデータ量を減らして再度試行
  • ネットワーク接続を確認
  • しばらく待ってから再試行

問題：ログインに失敗する

対処法：

  • メールアドレスとパスワードが正しいか確認
  • 会社IDが必要な場合は正しく入力しているか確認
  • JobcanのWebサイトで直接ログインできるか確認
