📖 勤怠管理・Jobcan用語集

勤怠管理とJobcan AutoFillに関連する用語を分かりやすく解説します。

勤怠管理の基本用語

勤怠管理（きんたいかんり）
Attendance Management
従業員の出勤・退勤時刻、労働時間、休暇取得などを記録・管理することです。労働基準法により、企業には従業員の労働時間を適正に把握する義務があります。
打刻（だこく）
Time Stamping
出勤・退勤の時刻を記録することです。従来はタイムカードを使用していましたが、現在はWeb上やICカード、生体認証などで行うことが一般的です。
例：朝9時に出勤打刻をする
出勤時刻（しゅっきんじこく）
Clock In Time
従業員が勤務を開始した時刻のことです。始業時刻と異なる場合、遅刻または早出勤となります。
退勤時刻（たいきんじこく）
Clock Out Time
従業員が勤務を終了した時刻のことです。終業時刻を過ぎている場合は残業となります。
労働時間（ろうどうじかん）
Working Hours
従業員が実際に労働した時間のことです。出勤時刻から退勤時刻までの時間から、休憩時間を差し引いたものです。労働基準法では、1日8時間、週40時間を超えた労働は原則として割増賃金の対象となります。
残業（ざんぎょう）
Overtime Work
所定労働時間を超えて働くことです。法定労働時間（1日8時間）を超えた場合、割増賃金が発生します。
遅刻（ちこく）
Late Arrival
始業時刻に遅れて出勤することです。遅刻時間分の給与が控除される場合や、評価に影響する場合があります。
早退（そうたい）
Early Leave
終業時刻より前に退勤することです。私用や体調不良などの理由で、通常より早く業務を終了することを指します。
欠勤（けっきん）
Absence
勤務日に出勤しないことです。事前に届け出る場合と、当日連絡する場合があります。無断欠勤は重大な就業規則違反となります。
有給休暇（ゆうきゅうきゅうか）
Paid Leave / Paid Time Off
給与が支払われる休暇のことです。労働基準法により、一定期間勤続した従業員には年次有給休暇が付与されます。

Jobcan関連用語

Jobcan（ジョブカン）
Jobcan
株式会社DONUTSが提供する勤怠管理システムです。Web上で出退勤の打刻、勤怠データの管理、シフト作成などが行えます。多くの企業で導入されています。
勤怠データ（きんたいデータ）
Attendance Data
出勤時刻、退勤時刻、労働時間、休暇取得状況などの記録データです。給与計算や労務管理に使用されます。
打刻修正（だこくしゅうせい）
Time Stamp Correction
間違って打刻した時刻や、打刻忘れを後から修正することです。通常は申請・承認が必要です。

Jobcan AutoFill関連用語

自動入力（じどうにゅうりょく）
Automatic Data Entry
人間が手動で行っていた入力作業を、プログラムが自動的に実行することです。本サービスの中核機能です。
テンプレートファイル
Template File
あらかじめ日付が入力されたExcelファイルのことです。開始時刻と終了時刻を入力するだけで使用できるため、手間を大幅に削減できます。
Excel形式（エクセルけいしき）
Excel Format
Microsoft Excelで作成・編集できるファイル形式です。本サービスでは.xlsx または .xls形式に対応しています。
アップロード
Upload
ローカルコンピュータのファイルをサーバーに送信することです。本サービスでは、勤怠データを記入したExcelファイルをアップロードして処理を実行します。
処理状況（しょりじょうきょう）
Processing Status
自動入力の進行状況を示す情報です。ログイン中、データ入力中、完了などの状態がリアルタイムで表示されます。
ログイン認証（ログインにんしょう）
Login Authentication
ユーザーの身元を確認するプロセスです。メールアドレスとパスワードを使用して、正当なユーザーであることを証明します。

IT・セキュリティ用語

HTTPS（エイチティーティーピーエス）
HyperText Transfer Protocol Secure
暗号化されたHTTP通信のことです。データの盗聴や改ざんを防ぎ、安全に情報をやり取りできます。本サービスではHTTPSを使用しています。
画像認証（がぞうにんしょう）
CAPTCHA
自動プログラム（ボット）ではなく、人間が操作していることを確認するための仕組みです。歪んだ文字や画像を選択することで証明します。
Cookie（クッキー）
Cookie
Webサイトがブラウザに保存する小さなデータファイルです。ログイン状態の保持や、ユーザーの設定を記憶するために使用されます。
キャッシュ
Cache
一度アクセスしたデータを一時的に保存しておく仕組みです。次回アクセス時に高速に表示できますが、古いデータが表示される場合もあります。
タイムアウト
Timeout
処理が一定時間内に完了しなかった場合、自動的に中断されることです。ネットワーク接続が不安定な場合などに発生します。

労働法関連用語

労働基準法（ろうどうきじゅんほう）
Labor Standards Act
労働条件の最低基準を定めた法律です。労働時間、休憩、休日、賃金などについて規定しています。
法定労働時間（ほうていろうどうじかん）
Statutory Working Hours
労働基準法で定められた労働時間の上限です。原則として1日8時間、週40時間までです。これを超えた労働には割増賃金が必要です。
36協定（さぶろくきょうてい）
Article 36 Agreement
労働基準法第36条に基づく協定です。これにより、法定労働時間を超えた時間外労働や休日労働が可能になります。
就業規則（しゅうぎょうきそく）
Work Rules / Employment Regulations
企業が定める労働条件や職場のルールです。労働時間、賃金、休暇、懲戒処分などについて規定されています。
